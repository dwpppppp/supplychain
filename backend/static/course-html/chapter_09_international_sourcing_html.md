<section id="section-1">

    <span class="section-num">Section 01</span>
    第一节 国际供应源搜寻概况
  </h2>

  <h3 class="subsection-title">一、国际供应源搜寻的概念</h3>

  <p>国际供应源搜寻是指一个国家从海外或其他国家的供应商处获取商品和服务的行为，也是为商品、服务或工程项目寻找外部供应商的过程。</p>

  <p>随着运输技术、信息通信技术的进步，贸易壁垒降低，采购搜索效率提高，企业能够通过更广阔的国际供应商网络寻找合适资源，并建立具有灵活性和竞争性的国际化供应网络。</p>

  <h3 class="subsection-title">二、促进国际供应源搜寻发展的因素</h3>

  <p>近年来，国际供应源搜寻快速发展，背后原因包括运输和通信技术进步、贸易壁垒降低、特定国家或地区的供应优势，以及跨境电子商务、国际营销服务网络和贸易促进平台的发展。</p>

  <ul class="info-list">
    <li>运输和通信技术提高了跨国采购效率。</li>
    <li>贸易壁垒降低使跨境采购更加便利。</li>
    <li>不同国家或地区拥有特定资源、成本或技术优势。</li>
    <li>跨境电商和国际服务网络降低了国际寻源门槛。</li>
    <li>时间和速度成为企业市场竞争力的重要因素。</li>
  </ul>

  <h3 class="subsection-title">三、国际供应源搜寻带来的优点和缺点</h3>

  <p>国际供应源搜寻是国际贸易的重要组成部分，其发展能够促进企业获取全球资源，也能推动国际贸易规模扩大。但它同时伴随价格、质量、管理和风险控制方面的复杂问题。</p>

  <h3 class="subsection-title">四、国际供应链搜寻的挑战</h3>

  <p>国际供应链搜寻面临物流、法规、海关、语言、文化差异、时区、货币波动、质量标准和沟通等挑战。同时，自然灾害、政治经济稳定性、脱碳和可持续性要求也可能影响国际供应源的稳定性。</p>

<section id="section-2">
  <h2 class="section-title">
    <span class="section-num">Section 02</span>
    第二节 国际供应源搜寻注意事项
  </h2>

  <p>随着全球贸易自由化程度不断加深，企业越来越倾向于在全球范围内寻找最佳供应商，以采购品质优良、价格合理的物料，实现资源有效配置。</p>

  <h3 class="subsection-title">一、国际供应源搜寻的具体内容</h3>

  <p>国际采购是根据既定标准寻找、评估和聘用供应商的过程。为了实现成本节约和组织价值最大化，企业需要在全球范围内寻找合适供应商。</p>

  <ol class="steps-list">
    <li>
      <strong>建立团队</strong>
      <p>组建具有采购、技术、财务、法务和物流能力的跨职能团队。</p>
    </li>
    <li>
      <strong>市场研究</strong>
      <p>分析目标国家和地区的供应市场、供应商资源和贸易环境。</p>
    </li>
    <li>
      <strong>战略开发</strong>
      <p>确定国际寻源目标、风险控制策略和供应商选择标准。</p>
    </li>
    <li>
      <strong>供应商选择</strong>
      <p>对潜在供应商进行资格审查、评估、谈判和筛选。</p>
    </li>
    <li>
      <strong>合同管理</strong>
      <p>明确贸易术语、价格、质量、交付、付款和争议解决条款。</p>
    </li>
  </ol>

  <h3 class="subsection-title">二、国际供应源搜寻注意事项</h3>

  <p>在全球范围内选择供应商比国内采购更复杂。采购方需要综合考虑供应商能力、政治环境、汇率、价格、质量、物流、法律、语言和文化等因素。</p>

  <h3 class="subsection-title">三、国际供应源评估</h3>

  <p>采购方可以运用一系列工具和技能评估国际供应商能力和风险，包括额外成本、贸易壁垒、支付风险、运输风险和污染风险等。评估过程中应记录所有与成本相关的风险，避免只比较表面报价。</p>

  <h3 class="subsection-title">四、国际供应源风险评估</h3>

  <p>国际供应源风险管理包括监测和控制供应商潜在风险，并创建具有多种灵活供应商选择的供应链。企业还需要识别可能影响供应获得的自然或人为灾害，制定全球供应源风险管理举措。</p>

  <h3 class="subsection-title">五、国际供应源风险管理技术</h3>

  <p>国际供应源风险管理技术包括识别和记录风险、构建供应源风险管理框架、监控风险，并实施降低风险、转移风险和接受风险的战略。</p>

  <ul class="info-list">
    <li>分析供应链每个流程环节、参与者和环境。</li>
    <li>识别风险事件的特征、原因、关系和潜在后果。</li>
    <li>建立多供应源或替代供应源机制。</li>
    <li>通过合同、保险和金融工具转移部分风险。</li>
    <li>持续监控政治、物流、汇率和质量风险。</li>
  </ul>

<section id="section-3">
  <h2 class="section-title">
    <span class="section-num">Section 03</span>
    第三节 《国际贸易术语解释通则》在国际采购中的应用
  </h2>

  <h3 class="subsection-title">一、《国际贸易术语解释通则》的介绍</h3>

  <p>《国际贸易术语解释通则》（Incoterms）由国际商会制定，是适用于国内与国际贸易的术语规则，有助于便利全球贸易并降低贸易风险。</p>

  <p>在货物销售合同中援引 Incoterms，可以在风险、费用、运输、海关清关安排等方面清晰界定买卖双方义务，从而减少潜在法律纠纷。</p>

  <p>Incoterms 并非法律强制要求，买卖双方可以自行约定合同术语。但如果合同明确采用 Incoterms，双方就应遵循相应版本规则中对风险和义务的规定。</p>

  <h3 class="subsection-title">二、Incoterms 的基本结构</h3>

  <figure>
    <figcaption><span class="fig-num">表 9-3</span>Incoterms 的四个分组</figcaption>

        <tr><td>CIF</td><td>成本、保险费加运费</td><td>装运港的运输工具上</td></tr>
      </tbody>
    </table>
  </figure>

  <p>EXW、FCA、CPT、CIP、DAP、DPU 和 DDP 适用于任何运输方式，能够满足多式联运需要。FAS、FOB、CFR 和 CIF 则仅适用于海运或内河运输。</p>

  <div class="supp-box">
    <div class="supp-box-header">
      <div class="supp-icon">补</div>
      <div class="supp-box-title">补充资料：常见 Incoterms 术语理解</div>
    </div>

    <ul class="info-list">
      <li>EXW 下卖方责任最小，买方承担较多运输、出口和风险责任。</li>
      <li>FCA 适用于多种运输方式，尤其适合集装箱多式联运。</li>
      <li>CIF 与 CFR 的区别在于卖方还需为买方投保货物运输风险。</li>
      <li>DPU 是唯一要求卖方在目的地卸货的术语。</li>
      <li>DDP 下卖方承担较多责任，包括进口清关费用和将货物运至指定目的地的风险。</li>
    </ul>
  </div>

  <div class="case-box">
    <div class="case-box-header">
      <span class="case-tag">案例</span>
      <div class="case-title">“全球寻源”在行动：寻源能力待提高</div>
    </div>

    <p>某石化公司计划员通过网上搜索发现，美国供应商所供配件在德国也有同类资源。由于美国供应商在中国市场处于垄断地位，价格长期难以下降。经过与德国供应商交流和比价，采购方发现两者差价巨大。</p>

    <p>采购方建议引进德国配件后，美国供应商为保住供应地位最终降低价格。该案例说明，在全球化采购背景下，采购部门的国际寻源能力和采购技巧直接关系到企业经济利益。</p>

    <p>提升全球寻源能力需要系统性培训、国际资源网络建设、跨区域采购组织协同，以及利用电子邮件、计算机辅助设计系统、条形码系统等工具降低沟通障碍。</p>
  </div>

  <div class="supp-box">
    <div class="supp-box-header">
      <div class="supp-icon">补</div>
      <div class="supp-box-title">补充资料：寻找国际供应源的渠道</div>
    </div>

    <p>因特网和贸易展览会是寻找国际货源的两大途径。展览会便于采购方直接接触样品和厂家，而因特网具有信息更新快、联系便利和范围广等优势。</p>

    <p>通过网络寻找货源时，图片不能替代实物样品。采购方应要求供应商提供样品，并在确认质量、价格、交期、包装、运输、保险、关税、沟通成本和库存成本后再作决策。</p>
  </div>

<section id="section-4">
  <h2 class="section-title">
    <span class="section-num">Section 04</span>
    第四节 实践指导
  </h2>

  <h3 class="subsection-title">一、实践任务</h3>

  <p>通过本章学习，学生需要学会在不同国际采购情境下分析和使用 Incoterms 条款，并通过案例分析加深对国际供应源搜寻、国际采购风险和国际贸易规则的理解。</p>

  <h3 class="subsection-title">四、实践内容</h3>

  <p>Incoterms2020 对各种贸易术语条件下买卖双方的基本义务作出了明确规定。在实际进出口业务中，选择哪一种贸易术语，需要根据具体交易情况决定，既要促进交易达成，也要避免承担过大风险。</p>

  <p>Incoterms 是国际贸易惯例，并非新版本自动取代旧版本。买卖双方在签订合同时，应注明采用的贸易术语和通则版本。若合同条款与惯例存在冲突，只要不违反适用法律，通常以合同约定为准。</p>

  <h3 class="subsection-title">五、实践范例</h3>

  <div class="case-box">
    <div class="case-box-header">
      <span class="case-tag">实践范例</span>
      <div class="case-title">共享公共采购市场的未来</div>
    </div>

    <p>政府采购市场是国内市场的重要组成部分。随着中国经济持续增长和政府采购改革深化，采购规模将持续扩大，也将为国内外企业带来更多商机。</p>

    <p>中国政府致力于建立公平竞争、公开透明的政府采购体系，保障在中国境内注册企业公平参与政府采购，并通过统一信息发布和供应商救济制度保护合法权利。</p>

    <p>中国加入世界贸易组织《政府采购协定》（GPA）将意味着中国与参加方互惠开放政府采购市场。对中国和 GPA 参加方而言，这都有助于实现更高水平的国际市场竞争和共赢发展。</p>
  </div>

  <div class="supp-box">
    <div class="supp-box-header">
      <div class="supp-icon">补</div>
      <div class="supp-box-title">前沿研究：信用证交易风险</div>
    </div>

    <p>信用证是国际贸易中常见的付款机制，能够为买卖双方提供一定安全保障。但信用证也可能被用于欺诈，例如提交表面符合信用证要求、实际与货物严重不符的单据。</p>

    <ul class="info-list">
      <li>进口商可能收到合规单据，却未必收到符合单据描述的货物。</li>
      <li>贸易商可能伪造单据或在单据中作欺诈性陈述。</li>
      <li>出口商可能因“不符点”遭遇开证行拒付。</li>
      <li>开证行或进口商可能无理拒付或无力支付。</li>
      <li>信用证中可能存在“软条款”，削弱其保证付款功能。</li>
    </ul>
  </div>

  <div class="case-box">
    <div class="case-box-header">
      <span class="case-tag">案例分析</span>
      <div class="case-title">如何拿下联合国订单</div>
    </div>

    <p>2016—2018 年，中国对联合国采购机构供应额占比较低。虽然中国医药和医疗器械出口规模较大，但在联合国采购和国际公立市场中的占比并不高。</p>

    <p>中国企业参与联合国采购面临多方面挑战，包括不熟悉国际采购规则、缺乏国际招投标专业人才、缺乏产品标准话语权、国际认证障碍较多、采购价格利润空间低、特定领域研发能力不足以及合同履约能力不足等。</p>

    <p>国际注册认证是产品进入国际市场的重要技术门槛。例如，世卫预认证已成为联合国机构和部分国际组织采购医药产品时的重要标准。企业若想进入国际公立市场，需要长期投入认证、质量体系和履约能力建设。</p>

    <div class="case-questions">
      <div class="case-questions-label">思考问题</div>
      <ol>
        <li>中国企业参与联合国采购面临哪些主要障碍？</li>
        <li>国际认证为什么会成为进入国际采购市场的重要门槛？</li>
      </ol>
    </div>
  </div>

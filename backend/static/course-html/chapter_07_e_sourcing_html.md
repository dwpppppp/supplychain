    <span class="section-num">Section 01</span>
    第一节 电子供应源搜寻的含义
  </h2>

  <h3 class="subsection-title">一、信息与通信技术的发展</h3>

  <p>信息与通信技术是信息技术与通信技术融合形成的概念。通信技术强调消息传播，信息技术强调信息编码、解码和在通信载体中的传输方式。随着技术进步，两者逐渐融合，并深刻影响采购活动的组织方式。</p>

  <p>网络通信技术的广泛应用为企业经营和日常工作带来巨大变化。买卖双方能够通过互联网实时沟通，提高采购信息处理速度，并以更低限制获取更广泛的市场知识和供应信息。</p>

  <p>信息与通信技术还推动了全天候采购商务。互联网采购突破了国际时区和地理距离限制，使采购方和供应商能够持续开展协作，同时支持虚拟团队、虚拟组织和供应链关系开发。</p>

  <h3 class="subsection-title">二、互联网的使用对采购成本的作用</h3>

  <p>互联网是一个全球性计算机网络，使计算机能够通过电信连接进行通信。互联网对商业活动的重要影响之一，是降低交易成本和互动成本，尤其是采购方寻找合适供应商时产生的搜寻成本。</p>

  <ul class="info-list">
    <li>整合和处理采购信息，统一向供应商订货，从而争取更大批量折扣。</li>
    <li>连接生产、库存和采购系统，实现实时订购，降低库存和资金占用。</li>
    <li>推动库存和订购管理自动化，减少人为干预和不必要损失。</li>
    <li>促进与供应商信息共享，使供应商按采购方生产需要进行供给。</li>
  </ul>

  <h3 class="subsection-title">三、电子采购和电子供应源搜寻</h3>

  <p>电子采购是由采购方发起的网上采购行为，包括网上招标、网上竞标和网上谈判等。它不仅完成采购交易，还利用信息技术和网络技术对采购全过程进行管理。</p>

  <p>电子供应源搜寻是指使用互联网，对如何以及从哪里获得服务或产品进行决策和制定战略的行为，也可以理解为使用电子化工具执行供应源搜寻全过程。</p>

  <h3 class="subsection-title">五、电子供应源搜寻的优势</h3>

  <p>电子供应源搜寻能够提高采购效率，降低采购方和供应商的信息搜寻成本；同时，由于过程更加透明，也有助于提升采购质量和市场分析能力。</p>

  <p>电子供应源搜寻系统虽然具有较大潜力，但必须被嵌入完整采购战略中。它不能完全替代线下供应源搜寻，尤其是在复杂产品、长期合作关系和高风险采购场景中，仍需要专业判断和线下沟通。</p>

    <p>企业通过线上平台将采购寻源范围从广东省扩展到全国，从“货比三家”变为“货比三十家甚至上百家”。同时采用全网询价、封闭报价和反向竞价等方式，省去中间环节，综合降低采购成本。</p>

<section id="section-2">
  <h2 class="section-title">
    <span class="section-num">Section 02</span>
    第二节 电子请购系统
  </h2>

  <h3 class="subsection-title">一、电子请购</h3>

  <p>电子请购是指为了简化采购流程，从用户处获取请购单信息，并通过互联网系统将需求信息提供给相关请购人，告知其请购要求的过程。</p>

  <p>电子数据库通常包含采购材料和零部件的记录代码、描述、使用量记录、当前余额、近期请购价格和供应商信息。随着收货、发货和库存水平变化，系统会自动更新相关数据。</p>

  <p>当库存达到预设水平，或生产计划触发采购需求时，电子请购系统可以自动生成再订购订单，从而提高采购计划的及时性和准确性。</p>

  <h3 class="subsection-title">二、制订需求计划的系统</h3>

  <p>需求计划系统可以从源头控制采购方的采购需求和计划，使每一项采购需求量都有据可依，减少不合规的多采购或少采购行为。</p>

  <ul class="info-list">
    <li>资源计划集成系统，如 MRP、制造资源计划和 ERP，可将主生产计划与物料清单转换成按时间划分的净需求。</li>
    <li>设计与开发系统可分析新产品设计生产所需材料、部件和配料，并生成对应请购单。</li>
  </ul>

  <h3 class="subsection-title">三、电子销售点系统</h3>

  <p>电子销售点系统（EPOS）可以与库存管理系统连接，触发自动库存请购和补货。系统通过条形码、无线识别标签和销售终端记录销售量，并将产品成本结构和库存状态集中存储在数据库中。</p>

  <p>当产品售出时，系统自动调整库存状态；当库存水平降至预设再订货点时，系统生成补货通知，从而提高销售、采购和订单跟踪的效率与准确性。</p>

  <h3 class="subsection-title">四、电子订单</h3>

  <p>电子订单是在数字方式下签订的客户订单。订单管理系统可以跟踪与订单相关的所有信息和流程，包括输入、库存管理、订单履行和售后服务。</p>

  <p>电子订单履行包括接收、包装、装运和向客户交付产品等过程，还包括端到端订单可见性、存储库存、拣货、包装、配送和退货处理等环节。</p>

  <h3 class="subsection-title">五、电子合同</h3>

  <p>电子合同是双方或多方当事人通过电子信息网络，以电子形式达成的设立、变更或终止财产性民事权利义务关系的协议。它通常通过数据电文、电子邮件等方式明确双方权利义务。</p>

  <ul class="info-list">
    <li>可以高效修改标准合同条款。</li>
    <li>可以更好保护合同内容，提升保密性。</li>
    <li>可以将合同数据高度集成，便于在线访问、版本管理和供应商绩效管理。</li>
  </ul>

<section id="section-3">
  <h2 class="section-title">
    <span class="section-num">Section 03</span>
    第三节 电子拍卖
  </h2>

  <h3 class="subsection-title">一、电子拍卖简介</h3>

  <p>电子拍卖是供应商使用电子手段显示可量化、涉及估价标准的投标要素新价格的过程。电子拍卖平台包括网站、电子屏幕、移动应用程序和短信平台，投标人可以通过互联网在线投标。</p>

  <div class="sourcing-overview">
    <div class="sourcing-card">
      <div class="num">01 / 正向拍卖</div>
      <h5>采购方竞买</h5>
      <p>供应商在线展示商品，潜在采购方出价竞买，出价最高者胜出。</p>
    </div>
    <div class="sourcing-card">
      <div class="num">02 / 反向拍卖</div>
      <h5>供应商竞价</h5>
      <p>采购方提出需求，供应商在线降价竞争，满足要求的较低报价获得订单。</p>
    </div>
  </div>

  <h3 class="subsection-title">二、电子拍卖的优缺点</h3>

  <p>电子拍卖在采购活动中的应用越来越普遍。它能够简化采购流程、节省时间，并提供在线比较价格的机会，同时突破地理限制，使用方便快捷。</p>

  <h4 class="minor-title">反向电子拍卖的优势</h4>

  <ul class="info-list">
    <li>减少供应商耗时，缩短请购前置期。</li>
    <li>利用竞争机制帮助采购方获得更低价格。</li>
    <li>促使现有供应商调整报价以保持竞争力。</li>
    <li>帮助采购方接触更多优质供应商并获取市场信息。</li>
    <li>减少采购方时间和精力投入，提高采购效率。</li>
    <li>适合保质期短或快速贬值产品和服务的采购。</li>
  </ul>

  <p>反向拍卖并非适用于所有采购情形。对于规格明确、供应商差异对采购方影响较小的产品，反向拍卖更容易发挥价格竞争优势；对于复杂产品或需要大量协作和谈判的项目，则不宜过度依赖反向拍卖。</p>

  <h4 class="minor-title">电子拍卖的主要缺点</h4>

  <ul class="info-list">
    <li>买卖双方缺乏面对面互动，可能产生信任问题。</li>
    <li>采购方可能失去信誉良好的供应商。</li>
    <li>低价中标供应商可能无法履行应尽义务。</li>
    <li>商品质量可能与平台展示或采购方预期不一致。</li>
  </ul>

  <div class="supp-box">
    <div class="supp-box-header">
      <div class="supp-icon">补</div>
      <div class="supp-box-title">补充资料：疫情背景下的网络拍卖</div>
    </div>

    <p>网络拍卖已广泛应用于房地产、机动车、股权、收藏品和艺术品等资产交易。新冠疫情使线下拍卖活动受到影响，许多传统拍卖企业将业务转移到线上。</p>

    <p>相较于传统拍卖，线上拍卖具有时间短、频率高、效率快、机动灵活等特点，是特殊环境下拍卖企业维持运营的重要方式。</p>
  </div>

<section id="section-4">
  <h2 class="section-title">
    <span class="section-num">Section 04</span>
    第四节 电子招投标
  </h2>

  <h3 class="subsection-title">一、电子招投标简介</h3>

  <p>电子招投标是以数据电文形式完成的招标投标活动，借助计算机和网络实现采购供应链能力。它基于网络通信过程，具有较强竞争性。</p>

  <ol class="steps-list">
    <li>
      <strong>提出投标邀请</strong>
      <p>招标邀请发布在采购方电子招标网站，供潜在供应商浏览或下载文件。</p>
    </li>
    <li>
      <strong>供应商响应</strong>
      <p>供应商通过安全电子邮件或系统提交标书，并注册为投标者。</p>
    </li>
    <li>
      <strong>数据分析与答疑</strong>
      <p>采购方通过前端网站分析投标情况，并对供应商查询进行回应。</p>
    </li>
    <li>
      <strong>在线开标评标</strong>
      <p>评标团队在线公开标书，并通过协作方式开展评标。</p>
    </li>
    <li>
      <strong>系统评分</strong>
      <p>系统对电子投标文件部分内容进行自动打分和评估。</p>
    </li>
    <li>
      <strong>结果通知</strong>
      <p>系统向中标者和未中标者发送评标结果并给予反馈。</p>
    </li>
  </ol>

  <h3 class="subsection-title">二、电子招投标的优势</h3>

  <ul class="info-list">
    <li>能够获取大量数据和指标。</li>
    <li>有助于创建公开、公平、公正的市场竞争秩序。</li>
    <li>降低管理成本和交易成本。</li>
    <li>优化招标代理机构内部资源利用。</li>
    <li>规范工程建设和采购流程。</li>
    <li>支持远程评标，提高评标组织效率。</li>
  </ul>

  <h3 class="subsection-title">三、电子招投标的缺点</h3>

  <p>电子招投标的主要问题包括缺乏统一技术标准和数据接口，容易形成信息孤岛，使数据库之间的信息难以共享，从而降低电子投标优势。</p>

  <ul class="info-list">
    <li>系统建设和维护成本较高。</li>
    <li>处理流程可能较长。</li>
    <li>发现违规行为的难度增加。</li>
    <li>竞争加剧后，投标人成功机会可能降低。</li>
  </ul>

  <h3 class="subsection-title">四、电子招投标的注意事项</h3>

  <p>采购方应设计标书以减少投标操纵，遵循密封投标涉及的步骤与注意事项，识别电子招投标中的危险信号，并管理电子投标环境中的新型欺诈风险。</p>

  <h3 class="subsection-title">五、电子招投标的最佳实践</h3>

  <ul class="info-list">
    <li>审查所有投标文件和附录。</li>
    <li>尽早提交投标响应，降低系统延误风险。</li>
    <li>设计在线投标响应时牢记最终目的。</li>
    <li>优化投标公告和登录页面。</li>
    <li>测试投标门槛，利用数字解决方案提升竞争力。</li>
  </ul>

  <div class="case-box">
    <div class="case-box-header">
      <span class="case-tag">案例</span>
      <div class="case-title">四川省将建统一电子招标和采购平台</div>
    </div>

    <p>四川省曾提出建设全省统一的电子招标采购平台，逐步实现网上投标、电子辅助评标和异地远程评标，推动通用货物政府采购实现“全覆盖、全流程、全上网、全透明”。</p>

    <p>该方案还提出建立统一公共资源交易服务平台、招标采购信用记录平台和违法行为记录公告平台，通过信息公开、信用共享和“一地受罚、处处受制”的机制强化招投标监督。</p>

    <div class="case-questions">
      <div class="case-questions-label">思考问题</div>
      <ol>

        <li>信用共享机制如何提升招投标监管效果？</li>
      </ol>
    </div>
  </div>

  <h3 class="subsection-title">四、实践内容</h3>

  <div class="case-box">
    <div class="case-box-header">
      <span class="case-tag">实践案例</span>
      <div class="case-title">利用电商平台采购航天物资的探索</div>
    </div>

    <p>中国运载火箭技术研究院物流中心负责全院物资采购、质量检验、仓储配送和供应链服务。随着航天事业发展，原材料种类、数量和质量要求持续提升，其中相当一部分为进口物资。</p>

    <p>在传统模式下，进口原材料、元器件和设备仪器主要通过合格进口物资中间商询比价、框架协议或公开招标方式采购。这种间接采购模式在价格、质量和信息资源方面存在风险。</p>

    <h4 class="minor-title">主要困难</h4>
    <ul class="info-list">
      <li>中间商环节增加采购成本，并可能导致价格被动接受。</li>
      <li>询价、谈判、合同签订和海外供货周期较长，影响科研生产进度。</li>
      <li>产品质量和运输仓储过程难以完全追溯。</li>
      <li>国外原厂产能、价格走势和新产品信息存在滞后或失真。</li>
    </ul>

  <h3 class="subsection-title">五、实践范例</h3>

  <div class="case-box">
    <div class="case-box-header">
      <span class="case-tag">实践范例</span>
      <div class="case-title">湖北政府采购实现全程电子化</div>
    </div>

    <p>湖北省交易采购中心完成省福利彩票发行中心视频型彩票终端机专用设备采购项目的网上开标和在线评审，标志着升级后的政府电子采购平台实现“不见面”开标和政府采购执行环节全流程电子化。</p>

    <p>升级后的平台实现公开招标、单一来源采购等六种政府采购方式的全流程电子化，并与政府采购管理系统、政府采购网和专家库数据互通，实现资料无纸化和“一键归档”。</p>

    <p>该平台还具备远程异地评审、人脸识别和大数据分析等功能，有助于提高采购效率、降低交易成本，并促进公平、公开、公正。</p>
  </div>

  <div class="supp-box">
    <div class="supp-box-header">
      <div class="supp-icon">补</div>
      <div class="supp-box-title">前沿研究：联合国电子招标要求</div>
    </div>

    <p>联合国项目事务署的电子招标可以通过电子邮件、电子采购系统或电子逆向拍卖系统进行。电子投标只要得到授权代表认可，并以规定格式提交，即具有法律约束力。</p>

    <p>投标截止时间同样适用于纸质和电子投标。投标人需确保投标文件在截止时间前被系统或专用邮箱收到，并在提交前检查电子文件病毒和可读性。</p>
  </div>

  <div class="case-box">
    <div class="case-box-header">
      <span class="case-tag">案例分析</span>
      <div class="case-title">“互联网+”背景下家电制造业采购模式的优化选择</div>
    </div>

    <p>在互联网和数字化经济背景下，传统采购模式难以满足企业采购需求。案例中的 L 公司存在交货准时性不稳定、供应商选择以价格为主、供应商控制力不足和单据审核效率低等问题。</p>

    <h4 class="minor-title">采购业务智慧化</h4>
    <ul class="info-list">
      <li>供应商入驻平台并展示商品、价格、资质和质量证明。</li>
      <li>企业发布招标公告，通过价格、数量、信誉等关键词智能筛选供应商。</li>
      <li>平台自动统计订单成本，为采购预算和成本控制提供数据。</li>
      <li>订单、单据和编码信息在线共享，降低沟通错误和审核成本。</li>
    </ul>

    <h4 class="minor-title">供应商管理一体化</h4>
    <ul class="info-list">
      <li>前馈控制：通过资质审核和综合评价筛选供应商。</li>
      <li>事中控制：采购流程可视化，订单状态可跟踪。</li>
      <li>反馈控制：定期评价供应商，实施分级管理。</li>
      <li>集成化服务：促进信息流、资金流和物流融合，降低牛鞭效应。</li>
    </ul>

        <li>未来 L 公司还可以在哪些方面继续优化采购模式？</li>
      </ol>
